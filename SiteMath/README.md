Bem vindo ao projeto AjudaMath! A ideia é fazer uma série de programas para auxiliar nas mais diversas áreas de matemática, desde algebra até tabuada por exemplo. Seja para fazer exercícios ou responder questões, o intuito é tornar mais fácil essas tarefas. Para tal, tem-se alguns executáveis: 

1. **RespondeMath:** Este será responsável por responder algumas questões relacionadas à área escolhida pelo usuário.

2. **ExerciMath:** Aplicação voltada para entrega de exercícios aos usuários, que ganharão pontos por acertar as questões.

3. **SiteMath:** Este site terá os dois anteriores, além de conteúdo teórico para incorporar as ferramentas.
