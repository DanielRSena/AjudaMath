# AjuMat - RespondeMat
Bem vindo ao AjuMat! A ideia é fazer uma série de programas para auxiliar nas mais diversas áreas de matemática, desde algebra até tabuada por exemplo. Seja para fazer exercícios ou responder questões, o intuito é tornar mais fácil essas tarefas. Para tal, tem-se alguns executáveis: 

1. **RespondeMat:** Este será responsável por responder algumas questões relacionadas à área escolhida pelo usuário. P-se pegar o código ou o .exe
2. **ExerciMat:** Aplicação voltada para entregar exercícios para os usuários, que ganharão pontos por acertar as questões.
3. **SiteMat:** Este site terá os dois anteriores, além de conteúdo teórico para incorporar as ferramentas.
