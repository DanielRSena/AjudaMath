# Muito bem vindo ao AjudaMath! 

A ideia é fazer uma série de programas para auxiliar nas mais diversas áreas de matemática, desde álgebra até tabuada por exemplo. Seja para
fazer exercícios ou responder questões, o intuito é tornar mais fácil essas tarefas. Para tal, tem-se alguns softwares: 

1. **RespondeMath:** Responde algumas questões relacionadas à área escolhida pelo usuário.

2. **ExerciMath:** Entrega exercícios para os usuários, que ganharão pontos por acertar as questões.

3. **SiteMath:** Este <em>site</em> terá os dois anteriores, além de conteúdo teórico.
