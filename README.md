# O repositório ferramentas matemáticas

A ideia é fazer uma série de programas para auxiliar nas mais diversas áreas de matemática, desde algebra até tabuada por exemplo. Seja para fazer exercícios ou responder questões, o intuito é tornar mais fácil essas tarefas. Para tal, tem-se alguns executáveis: 

1. **AjuMat:** Este será responsável por responder algumas questões relacionadas à área feita pelo usuário
2. **ExerciMat:** Aplicação voltada para entregar exercícios para os usuários, que ganharão pontos por acertar as questões
