 # Bem vindo!!


A ideia é fazer uma série de softwares para auxiliar nas mais diversas áreas de matemática, desde álgebra até tabuada por exemplo. Seja para fazer exercícios ou responder questões, o intuito é tornar mais fácil essas tarefas. Para tal, tem-se alguns programas: 

## RespondeMath
Esse programa em C responde algumas questões relacionadas à área escolhida pelo usuário. Os atuais recursos são:
1. Agrupamento discreto
2. Equação de 1º grau
3. Equação de 2º grau
4. Matrizes (soma, subtração, multiplicação por escalar e matriz transposta)
5. Sistemas lineares
6. Tabuada (0 até 10)

## ExerciMath
O programa é feito em Java exercícios para os usuários, que ganharão pontos por acertar as segintes questões:
1. Agrupamento discreto
2. Equação de 1º grau
3. Equação de 2º grau
4. Sistemas lineares
5. Tabuada contra o tempo (0 até 10, 15 segundos)

## SiteMath
Este <em>site</em> tem conteúdo teórico e resolução dos seguintes temas:
1. Equação de 1º grau
2. Equação de 2º grau
3. Sistemas lineares
