# Fórmulas Matemáticas

 A ideia é ter um **programa geral** que consegue resolver e explicar várias funções e ferramentas matemáticas que podem ser úteis em algum momento. Ademais, terão as partes separadamente e exercícios gerados na hora para cada uma delas. Como projeto futuro, poderá haver um programa específico para cálculos estatísticos.
